【履歴】
2009/11/18	拗音（ゃゅょ等）に対応しました。
--------
【概要】

音声合成ツールUTAU用のプラグインツールです。
選択範囲の音符の連続音用の歌詞データ（「o お2」等）を単独音用（「お」）に変換します。

Windows Script Host + JScript + ラッパーアプリケーション（maidroさん作）を利用しています。
一般に、UTAUを起動できる環境であれば利用できるのではないかと思います。

--------
【使用方法】
アーカイブを解凍したフォルダを、そのままUTAUのpluginsフォルダに移動してください。
UTAU上でプラグインのリロードを行えば使用可能になります。
実行時、セキュリティソフトの警告が表示される場合がありますので、あらかじめご了承ください。
